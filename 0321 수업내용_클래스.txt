*소멸자: 당장 소멸시켜야할 때, 소멸되기 전 무언가 처리해야할 때


*init 코드 : 객체 생성
class Example:
   def __init__(self, name):
   self.name = name
   print(f"{self.name} 객체 생성됨!")


*del코드: 객체 삭제하는 코드
def __del__(self):                          #소멸되기 전 진행되어야 할 코드
   print(f"{self.name} 객체 소멸됨!")



obj=Example("테스트")    변수 = 클래스 이름("여기는 객체 이름")       #객체 생성 
del obj  #객체 삭제 -> 소멸자(del코드로 돌아가서) 실행
